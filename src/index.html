Get Youtube Subscribers

You can try entering each route using the routes given below

  • GET /subscribers : Provides an array of all subscribers details in the database.
  • GET /subscribers/names : Provides an array of all subscribers with only two fields, their names and subscribed channel.
  • GET /subscribers/:id : Provides the details of subscriber with the given ID.
  • POST /subscribers : create a new subscriber
  • POST /subscribers/names : create a new subscriber with names only
  • POST /subscribers/:id : create/update a new subscriber by ID
  • DELETE /subscribers/:id : delete a subscriber by ID
